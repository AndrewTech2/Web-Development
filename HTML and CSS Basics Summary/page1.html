HTML & CSS Basics Summary

HTML Summary

HTML is used to definde our page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses elements to describe content

    Most HTML elements have an opening tag, content and closing tag.

  • You can also have empty elements like images

  • You can configure elements with attributes

    There's a long list of available elements but you'll gain knowledge over time, so there is no need to worry.

You can learn more about all the availabe HTML elements on the MDN HTML element reference.

CSS Summary

CSS is used for styling your page content.

  1. Styles are assigned via property-value pairs

  2. You can assign styles via the 'style' attribute

  3. To avoid code duplication, you can use global styles instead

  4. Alternatively, you can work with external stylesheets files which you link in the HTML file

  5. When working with CSS, concepts like 'inheritance', 'specifity' and the 'box model' should be understood.

Learn more about all available CSS properties and values at the MDN CSS property reference.
